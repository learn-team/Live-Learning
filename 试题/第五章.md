###数组的 length 除了可以读取数组的长度，还有很多用法。请举例几种常见使用方法。

答：
1. 移除数组项

        var arr = [1,2,3,4,5];
        arr.length = 2; 
        console.log( arr );     //[1, 2]

2. 设置数组长度

            var arr = [1,2];
            arr.length = 4; 
            console.log( arr, arr.length );     //[1, 2]     4

3. 数组尾部添加新项

            var arr = [1,2,3];
            arr[ arr.length ] = arr.length; 
            console.log( arr );         //[1, 2, 3, 3]
